Need to find the Panorama ID of a Google StreetView manually? It's encoded in the StreetView URL between !1s and !2e. For example, this StreetView includes the id bolded in this string: data=!3m6!1e1!3m4!1srviXpriRVAKxt77Y3D9tzg!2e0!7i3328!8i1664.
